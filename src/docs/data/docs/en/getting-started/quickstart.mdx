# Quickstart Guide

Get your first Lumenize backend up and running in under 5 minutes. We'll build a simple task management API to demonstrate the core features.

## Prerequisites

- A [Lumenize account](/docs/hosted/signup)
- Basic knowledge of JSON and REST APIs

## Step 1: Create a New Project

1. Log into your [Lumenize dashboard](https://app.lumenize.dev)
2. Click **"New Project"**
3. Enter project details:
   - **Name**: "Task Manager"
   - **Description**: "A simple task management backend"
   - **Region**: Choose closest to your users
4. Click **"Create Project"**

## Step 2: Define Your Schema

Let's create a simple task schema:

```json
{
  "name": "Task",
  "fields": {
    "title": {
      "type": "string",
      "required": true,
      "maxLength": 200
    },
    "description": {
      "type": "text",
      "required": false
    },
    "completed": {
      "type": "boolean",
      "default": false
    },
    "priority": {
      "type": "enum",
      "values": ["low", "medium", "high"],
      "default": "medium"
    },
    "dueDate": {
      "type": "datetime",
      "required": false
    },
    "assignedTo": {
      "type": "reference",
      "entity": "User",
      "required": false
    }
  },
  "indexes": ["title", "completed", "priority"],
  "permissions": {
    "create": ["authenticated"],
    "read": ["owner", "assignedTo"],
    "update": ["owner", "assignedTo"],
    "delete": ["owner"]
  }
}
```

1. In your project dashboard, click **"Add Schema"**
2. Paste the schema above
3. Click **"Save & Deploy"**

## Step 4: Make Your First API Call

### Create a Task

```bash
curl -X POST https://api.lumenize.dev/v1/proj_abc123/tasks \
  -H "Authorization: Bearer lum_live_xyz789" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Build my first Lumenize app",
    "description": "Follow the quickstart guide",
    "priority": "high"
  }'
```

### Fetch All Tasks

```bash
curl https://api.lumenize.dev/v1/proj_abc123/tasks \
  -H "Authorization: Bearer lum_live_xyz789"
```

### Update a Task

```bash
curl -X PATCH https://api.lumenize.dev/v1/proj_abc123/tasks/task_123 \
  -H "Authorization: Bearer lum_live_xyz789" \
  -H "Content-Type: application/json" \
  -d '{"completed": true}'
```

## Step 5: Connect Your Frontend

### Using JavaScript

```javascript
import { LumenizeClient } from '@lumenize/client';

const client = new LumenizeClient({
  projectId: 'proj_abc123',
  apiKey: 'lum_live_xyz789'
});

// Create a task
const task = await client.tasks.create({
  title: 'Learn Lumenize',
  priority: 'high'
});

// Subscribe to real-time updates
client.tasks.subscribe((event) => {
  console.log('Task updated:', event);
});
```

### Using React

```jsx
import { useLumenize } from '@lumenize/react';

function TaskList() {
  const { data: tasks, create, update } = useLumenize('tasks');

  const addTask = async () => {
    await create({
      title: 'New task',
      priority: 'medium'
    });
  };

  return (
    <div>
      {tasks.map(task => (
        <div key={task.id}>
          <h3>{task.title}</h3>
          <button onClick={() => update(task.id, { completed: true })}>
            Complete
          </button>
        </div>
      ))}
      <button onClick={addTask}>Add Task</button>
    </div>
  );
}
```

## Step 6: Explore Real-time Features

Lumenize automatically provides WebSocket subscriptions for all your data:

```javascript
// Subscribe to all task changes
client.tasks.subscribe('*', (event) => {
  console.log(`Task ${event.id} was ${event.type}d`);
});

// Subscribe to specific task
client.tasks.subscribe('task_123', (event) => {
  console.log('This task was updated:', event.data);
});

// Subscribe with filters
client.tasks.subscribe({
  where: { completed: false },
  orderBy: 'dueDate'
}, (tasks) => {
  console.log('Pending tasks:', tasks);
});
```
