## Frequently Asked Questions

### What counts as an API request?

Every HTTP request to your Lumenize API counts as one request. This includes:
- REST API calls (GET, POST, PUT, DELETE)
- GraphQL queries and mutations
- WebSocket connections (counted per connection establishment)
- Real-time subscription events do not count as additional requests

### How does billing work?

- **Monthly billing** on your plan anniversary
- **Pay-as-you-go** for usage beyond plan limits
- **Annual discounts** available (20% off yearly plans)
- **No setup fees** or hidden charges

### Can I change plans anytime?

Yes! You can upgrade or downgrade your plan at any time:
- **Upgrades** take effect immediately
- **Downgrades** take effect at the next billing cycle
- **Prorated billing** for mid-cycle upgrades

### What happens if I exceed my limits?

- **API requests**: Automatically billed at add-on rates
- **Storage**: Graceful degradation with upgrade prompts
- **No service interruption** for temporary overages

### Do you offer refunds?

- **30-day money-back guarantee** for paid plans
- **Pro-rated refunds** for annual subscriptions
- **No refunds** for add-on usage charges

### Is there a free trial for paid plans?

- **14-day free trial** for Starter and Professional plans
- **No credit card required** to start trial
- **Full access** to all plan features during trial
